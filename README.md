# RAG Chatbot (Docs Q&A)

Upload documents and ask questions. The bot answers using retrieval-augmented generation (RAG) and provides citations.

## Features (MVP)
- Upload PDF/TXT/MD
- Build a local Chroma vector index
- Chat UI (Streamlit)
- Answers with sources/citations

## Quickstart
1. Create a virtual env
2. Install dependencies
3. Add your OpenAI key
4. Run Streamlit
